## Description

This module installs and configures [OpenVoxView](https://github.com/voxpupuli/openvoxview)

## Usage

```puppet
include openvoxview
```


Install specific version of OpenVox View

```puppet
class { 'openvoxview':
  version        => '0.1.12',
}
```
